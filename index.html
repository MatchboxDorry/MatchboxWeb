The IT platform customized for micro enterprises

Dorry is a professional enterprise platform, running on ARM and continually
delivering secure enterprise applications and services, which is based on the
leading technology of docker virtualization.

The Constructure of Dorry

Features

Security

Dorry Market provides application authentication and security validation.

Lightweight

Dorry OS entitles the application with the capabilities of instant bootstrap, less RAM and disk usage.

Simplicity

Running your enterprise services on Dorry OS as on your smartphone only with two neat steps “Install->Start”.

Integration and Management

Dorry OS integrates enterprise applications in one centralized platform with management capability.

Local deployment

Dorry OS is deployed on Raspberry Pi by default in the local enterprise environment, so are the enterprise services and data.

Functionalities

Update & Upgrade

DorryOS and enterprise applications can be updated and upgraded on the fly.

SSO

DorryOS provides Single-Sign-On to access to enterprise services.

Backup & Restore

DorryOS provides the service to backup and restore application data.

Micro Service Communication

Each executing enterprise application is considered as micro service. DorryOS handles the service communication.

Management

DorryOS provides user friendly interface to manipulate enterprise services on the fly.

Feedback

DorryOS reports application execution errors and exceptions.
